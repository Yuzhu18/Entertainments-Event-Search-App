# Entertainments-Event-Search-App

a Android App used for searching entertainments

[Video Demo](https://youtu.be/43F4Lu1xRls)

### Based on several APIs: 
* Ticketmaster API
* Google Map API
* Spotify API
* Songkick API
* Twitter API

### Developed on Android Studio && Deployed on AWS Elastic Beanstalk
